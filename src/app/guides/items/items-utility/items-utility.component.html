Utility Item Build Overview

This item set-up is great for playing the "Tradtional" Support role, offers more utility than damage. The build excels if your team has enough damage deals (very common in SoloQ) and you need to push the team over across the line with extra heals, shields and the inherent utility that Zilean has to offer.

This build should help you become more comfortable with using active items (if you are not used to it).

Recommended Runes
Starting Items

This is a pretty generic start that gives you a little extra power in lane. If you are playing against a lane that has little pressure, start Refillable Potion; otherwise start 3 Potions.

Early Game Items

Your first back should get you the upgraded gold item, then you'll want to build Sightstone and Boots. If you are having trouble sustaining mana, buy a Chalice of Harmony, it will build into Athene's Unholy Grail.

Mid Game Core Items

The objective is to reach max CDR (40%) as soon as possible. You have some options about which of these items you can complete first, Athenes' first is a well-rounded option because you can give extra healing from Summon Aery + Time Warp (on ally).

End Game Items (Full Utility)

This ending build has plenty of active items that work well in team fights. This build does play in the back-line protecting carries, so be sure to protect them first.

Optional Utility Items

These are defensive options that you can build depending on the enemy team composition. For instance, take Mikael's Crucible if the enemy team has an abundance of Crowd Control, or are looking for a single target to engage on (stopping the initial CC along with Time Warp should be enough to stop an enemy engage).
